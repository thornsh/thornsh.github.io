# Array.from()

将类数组，可迭代对象转换为真数组，等价于：Array.prototype.slice.call()

# concat()

该方法默认会打平数组

let colors = ["red", "green", "blue"];
let colors2 = colors.concat("yellow", ["black", "brown"]);

console.log(colors2); // ["red", "green", "blue", "yellow", "black", "brown"]

# Symbol.isConcatSpreadable

该符号为true时，打平数组，false时不打平数组

let colors = ["red", "green", "blue"];
let colors2 = ["black", "brown"];

colors2[Symbol.isConcatSpreadable] = false;

let newColor = colors.concat("yellow", colors2);

console.log(newColor); // ["red", "green", "blue", "yellow", ["black", "brown"]]

# WeakMap

WeakMap中表示弱映射的键是弱弱的拿着的。这些键不属于正式的引用，不会阻止垃圾回收。

const wm = new WeakMap();
wm.set({}, "val");
1
2

因为这个键没有指向对象的其他引用，所以代码执行完毕后，该对象键会被当成垃圾回收。

const wm = new WeakMap();

const container = {
    key: {}
};

wm.set(container.key, "val");

function removeReference() {
    container.key = null;
}
