# 对象方法

# 禁止扩展 Object.preventExtensions()

如果想禁止一个对象添加新属性并且保留已有属性，可以使用 Object.preventExtensions()

var myObject = {
  a: 2,
};
Object.preventExtensions(myObject);
myObject.b = 3;
myObject.b; // undefined

# 密封 Object.seal()

这个方法本质上调用 Object.preventExtensions()并把所有现有的属性标记为configurable: false

所以密封之后不能添加新属性，并且不能重新配置或者删除现有属性,但是可以修改属性的值

# 冻结 Object.freeze()

这个方法在密封的基础上把所有“数据访问”属性标记为writable: false，使得无法修改值

# propertyIsEnumerable()

检查给定的属性名是否直接存在于对象中，并且满足可枚举

# 存在性

# in 操作符

in 操作符会检查属性是否存在对象以及其[[Prototype]]原型链中。

# hasOwnProperty()

hasOwnProperty()则只会检查属性是否存在当前对象中，并不会检查原型链

# Object.keys()

只会查找当前对象，返回可枚举的属性数组

# Object.getOwnPropertyNames()

只会查找当前对象，返回所有属性数组，不管是否可枚举

TIP

因为 hasOwnProperty()是存在对象 prototype 上的方法，有些对象有可能没有连接到 Object.prototype,(比如通过 Object.create(null)来创建的对象)，这时候可以使用：Object.prototype.hasOwnProperty.call(myObject, "a")来判断是否有属性 a
