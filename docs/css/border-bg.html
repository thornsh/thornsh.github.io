# background-clip

默认情况下背景会延伸到边框所在区域下层，可以使用background-clip解决

background-clip默认值是border-box,设置为padding-box浏览器就会用内边距的外沿来把背景剪裁掉

# box-shadow实现多重边框

background: yellowgreen;
box-shadow: 0 0 0 10px #655, 0 0 0 15px deppink;

2

# outline实现两重边框

background: yellowgreen;
border: 10px solid #655;
outline: 5px solid deppink;

# outline-offset属性控制它和元素边缘之间的间距

# background-position

使用background-position指定背景图片距离任意角的偏移量

background: url(xxx.png) no-repeat #58a;
background-position: right 20px bottom 10px;
1
2

# background-origin

通过background-origin使背景图片偏移与容器内边距一致

div {
    padding: 10px;
    background: url('xxx.png') no-repeat #58 bottom right;
    background-origin: content-box;
}
1

默认情况下，background-origin的值是padding-box

# 结合使用实现图片边框

div {
    padding: 10px;
    border:10px solid transparent;
    background: linear-gradient(white, white), url(xxx.png);
    background-size: cover;
    background-clip: padding-box, border-box;
    background-origin: border-box;
}
