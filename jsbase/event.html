# el.addEventListener(eventName, handle, useCapture|options)

  • handle: 事件函数
  • useCapture：true代表捕获阶段触发，false代表冒泡阶段触发
  • options：
    • capture: 同useCapture
    • once：true代表事件监听只执行一次
    • passive：true代表事件不会调用preventDefault()

# el.removeEventListener(eventName, handle)

移除通过addEventListener()添加的事件处理函数

TIP

如果要移除，必须给addEventListener()函数的回调不是匿名函数

# event.currentTarget

与this的引用相同

# event.target

真正触发事件的元素

# event.eventPhase

1表示捕获阶段，2表示目标阶段，3表示冒泡阶段

# event.preventDefault()

阻止事件的默认行为，只有event.cancelable属性为true才能取消。不会阻止事件传播。在事件监听函数末尾提供return false也有相同效果

# event.stopPropagation()

阻止捕获和冒泡阶段中当前事件的进一步传播。

# event.stopImmediatePropagation()

阻止事件冒泡，且还能阻止执行该语句之后的所有事件监听，不阻止默认事件

# 自定义事件 document.createEvent()

<body>
    <div>click me</div>
    <script>
    	var div = document.querySelector('div');
        // 创建自定义事件
        // CustomEvent参数必须
        var cheer = document.createEvent('CustomEvent');
        
        // 创建自定义事件的监听
        div.addEventListener('goBigBlue', function(event) {
            console.log(event.detail.goBigBlueIs);
        }, false);
        // 使用initCustomEvent配置自定义事件详情
        // {event, bubble?, cancelable?, event.detail}
        // bubble:冒泡， cancelable：是否可以preventDefault
        cheer.initCustomEvent('goBigBlue', true, false, {
            goBigBlueIs: 'its gone!'
        });
        
        // 使用dispatchEvent触发自定义事件
        div.dispatchEvent(cheer);
    </script>
</body>

# 模拟触发鼠标事件

<body>
    <div>不用点击，会自行触发</div>
    <script>
    	var div = document.querySelector('div');
        div.addEventListener('click', function(event) {
            console.log(Object.keys(event));
        }, false);
        
        // 创建模拟的点击事件
        var divClick = document.createEvent('MouseEvents');
        
        // 配置
        divClick.initMouseEvent('click', true, true, document.defaultView, 0, 0, 0, 0, 0, false, false, false, 0, null, null);
        
        //触发
        div.dispatchEvent(divClick);
    </script>
</body>

# 事件委托

下例所示监听parent的四个子div元素的点击事件

<div class="parent">
    <div>
        <span>one</span>
    </div>
    <div>
        <span>two</span>
    </div>
    <div>
        <span>three</span>
    </div>
    <div>
        <span>four</span>
    </div>
</div>

<script>
	var parent = document.querySelector('.parent');
    parent.addEventListener('click', () => {
        var target = e.target;
        while(target !== e.currentTarget) {
            if(target.matches('div')) {
                // do something
            }
            target = target.parentNode;
        }
    })
</script>
